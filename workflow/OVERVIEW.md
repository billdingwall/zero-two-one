# Workflow Overview

Documentation defining the overall project workflow, personas involved.

## Manifest

- `workflows.md`: **Canonical expanded workflow reference** — Discovery, Design, Refinement, Speckit Implementation, QA, and Release, with agent/automation touchpoints and enforcement gates.
- `workflow-overview.md`: Original overview of the two-workflow model (superseded by `workflows.md` for process detail).
- `project-manifest.md`: Outlines the project's folder and document structure.
- `specific-workflows/`: Directory for specific workflow documentation.
- `_personas/`: Directory containing user, stakeholder, and contributor personas.
