# Product Lifecycle Checklist

This document tracks the operational state of the project through the 4-phase Zero Two One lifecycle.

## Phase 1: Planning (Zero)
- [ ] Initialize repository
- [ ] Fill out `requirements/01-PRD.md`
- [ ] Fill out `requirements/03-TDD.md`
- [ ] Fill out `requirements/04-ROADMAP.md`

## Phase 2: Pre-build (Refinement)
- [ ] Fill out `requirements/02-EDD.md`
- [ ] Define `DESIGN.md` tokens
- [ ] Build static prototype in `prototype/`
- [ ] Refine via review loops in `requirements/_refinement/`

## Phase 3: MVP Build (One)
- [ ] Generate Spec Kit artifacts in `specs/`
- [ ] Implement features against specs
- [ ] QA and accessibility validation
- [ ] MVP Release

## Phase 4: Growth
- [ ] Gather user feedback
- [ ] Feed data back into the refinement loop
- [ ] Ship enhancements via Spec Kit
