# Zero Two One

Welcome to Zero Two One. This project is designed to standup brand new projects starting at point 0 and taking them to point 1 (MVP) and helping them grow and stabilize from there.

The goal is to give product founders a solid starting point for building a new product based on the need they see, the role they want their product to fill, and the proposed functionality to fit that role and need.

## Getting Started

This repository provides a framework for project-level documentation, refinement workflows, and feature delivery using Github Speckit. 

To begin:
1. Initialize your AI assistant and prompt it to help you fill in the key documents located in the `requirements/` folder (e.g., `01-PRD.md`, `02-TDD.md`).
2. See **`MEMORY.md`** for the 4-phase project lifecycle and to track the current active phase. Update it along with `CLAUDE.md` and this `README.md` with your specific project details.
